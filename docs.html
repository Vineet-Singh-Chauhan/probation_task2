Install CLI

Requirements

  • Windows 10 or higher
  • Node.js v12 or higher
  • DirectX.12

Install

Mac (Hebrew)

$ brew install loruki-cli

NPM

$ npm install loruki-cli

yarn

$ yarn install loruki-cli
